# Customer Churn Prediction

## Overview

This project predicts whether a customer is likely to leave a company based on customer demographics, subscription details, and usage patterns.

## Technologies Used

- Python
- Pandas
- Scikit-learn
- Matplotlib

## Machine Learning Algorithm

- Random Forest Classifier

## Evaluation Metric

- Accuracy Score

## Dataset

Telco Customer Churn Dataset from Kaggle

## Features

- Customer demographics
- Subscription details
- Monthly charges
- Contract information

## Result

The model predicts customer churn and helps businesses identify customers at risk of leaving.
